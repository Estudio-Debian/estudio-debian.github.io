# ![](https://raw.githubusercontent.com/PapirusDevelopmentTeam/papirus-icon-theme/master/Papirus/48x48/apps/com.github.bartzaalberg.snaptastic.svg) Remover Snap

No Ubuntu, o Snap vem instalado e ativo por padrão, isso gera uma pequena perda de desempenho por conta do consumo de CPU e memória, para garantir que você possa extrair o máximo do seu hardware, é recomendado remover completamente esse suporte e não usar programas nesse formato.

Os scripts deste repositório fornecem outras formas de instalar programas além do `apt`.

Programas no formato snap também costumam ocupar mais espaço em disco.
