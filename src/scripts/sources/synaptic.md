# ![](https://raw.githubusercontent.com/PapirusDevelopmentTeam/papirus-icon-theme/master/Papirus/48x48/apps/synaptic.svg) Synaptic

O Synaptic é um gerenciador de pacotes _deb_, com ele você pode instalar, desinstalar, atualizar, etc.

Mas, a presença dele aqui se dá por um outro recurso pouco comentado, que é _resolver_ problemas com pacotes quebrados ou incompatíveis.

Com ele é possível mudar a versão de um pacote instalado, por exemplo, pra fazer um _downgrade_, se necessário.

Isso ajuda na hora de resolver problemas inesperados caso você mexa onde não deve.
