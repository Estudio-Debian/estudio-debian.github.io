# ![](https://raw.githubusercontent.com/PapirusDevelopmentTeam/papirus-icon-theme/master/Papirus/48x48/mimetypes/application-vnd.flatpak.svg) Flatpak

Flatpak é o modelo de empacotamenteo universal e provavelmente se tornará padrão em várias distribuições no futuro, embora eles tenham certa similaridade com snaps por conta do tamanho ocupado em disco, se comportam de forma mais fluída, além de ter uma variedade enorme de programas disponíveis neste formato.

Nenhum script deste repositório instala programas através do Flatpak.
