# ![](https://raw.githubusercontent.com/PapirusDevelopmentTeam/papirus-icon-theme/master/Papirus/24x24/apps/carla.svg) Estúdio Debian
![](src/static/studio.png)

O **Estúdio Debian** é um repositório de scripts cujo objetivo é configurar o Debian, o Ubuntu e seus derivados pra trabalhar com criação de conteúdo multimídia, com foco especial no áudio em baixíssima latência.

Além dessas configurações, os scripts também instalam diversos programas com configurações específicas pra otimizar o desempenho e a usabilidade.

Tudo aqui é testado exaustivamente pra entregar o melhor resultado possível.
